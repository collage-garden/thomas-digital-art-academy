# 知识

*工具架，不是课表。*

这里放的是学科知识——整理好了方便你找，但别误会，它们不是花园的目的。知识服务于创作，不是反过来。

你不是来这里"学物理"的。你来是因为手里在做东西，需要搞明白一根弦为什么那样振动。拿走你要的，回去继续干活。
