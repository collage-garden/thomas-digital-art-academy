## 作品

### 🎧 [双耳拍探索器](https://collage-garden.github.io/binaural-beats/)

一个跑在浏览器里的交互式音频体验。实时调频率、切分配策略、换模式，Canvas 画出拍频包络，还能拖个音乐文件进来做频移。工程上从一个裸的 `OscillatorNode` 演进到 AudioWorklet + Web Worker SSB 调制架构，[8 步记录](https://github.com/collage-garden/binaural-beats/tree/main/docs)在案。

纯前端，零依赖。Web Audio API、AudioWorklet、Canvas、PWA。
