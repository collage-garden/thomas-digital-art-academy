# Interactive Installation

*Where technology meets physical space and human participation.*

This path explores the creation of interactive experiences — installations that respond to presence, touch, sound, or movement. From sensor-driven art to generative environments, this is where engineering and art become inseparable.

---

## Works

### 🎧 [Binaural Beats Explorer](https://collage-garden.github.io/binaural-beats/)

A browser-based interactive audio experience. Real-time parameter control (frequency, distribution strategy, mode), beat envelope visualization on canvas, drag-and-drop audio upload for Music SSB mode. The engineering journey — from a bare OscillatorNode to AudioWorklet + Web Worker SSB modulation — is documented in 8 evolutionary steps.

Pure frontend, zero dependencies. Web Audio API, AudioWorklet, Canvas, PWA.
