# 花园小径

*六条路径，彼此交叉。*

它们不是平行跑道。一个项目完全可以同时长在两三条路径的交汇处。

---

### 🌿 [日常生活](everyday-life/)

藏在日常里的艺术和科学

### 🔭 [社会观察](social-observation/)

社会怎么运转，人怎么互动，文化怎么变

### 🎵 [听觉艺术](auditory-art/)

声音、音乐、声学，以及「听」本身

### 🎨 [视觉艺术](visual-art/)

图像、形态、色彩、空间，以及「看」本身

### 🕳️ [互动装置](interactive-installation/)

技术遇上物理空间和人的参与

### 🎭 [传统艺术](traditional-art/)

用今天的眼睛重新看老手艺
