# 听觉艺术

*关于声音、音乐、声学，以及「听」这件事。*

竹笛为什么那样响，和声背后有什么数学，音乐为什么能让人起鸡皮疙瘩——这条路径关注声音怎么被造出来、怎么被塑形、怎么被大脑接收和感受。

---

## 作品

### 🎧 [双耳拍探索器](https://collage-garden.github.io/binaural-beats/)

左耳一个频率，右耳另一个，大脑会「听」到一个物理上根本不存在的节拍。这个工具让你亲身体验这个现象，然后追问一个更难的问题：同样的声音换一套说辞，感受会不会完全变了？

三种音频模式从不同角度打开这个现象。三个讽刺版本（科学 / 营销 / 灵修）演示叙事怎样改变对同一段声音的感知。

**涉及知识**：物理（声学、波叠加） · 数学（FFT、希尔伯特变换） · 神经科学（上橄榄复合体） · 心理学（感知、安慰剂效应）

**也在**：[互动装置](../interactive-installation/) · [社会观察](../social-observation/) · [日常生活](../everyday-life/)
