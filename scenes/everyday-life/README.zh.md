# 日常生活

*藏在日常里的艺术和科学。*

物理、数学、心理学、设计——这些东西不是只在课本里才有意义，它们在你每天走过的路、用过的工具、听到的声音里无处不在。这条路径从日常出发，把平凡里的不寻常翻出来。

---

## 作品

### 🎧 [双耳拍探索器](https://collage-garden.github.io/binaural-beats/)

你天天戴耳机，但你知道左右耳听到的频率稍有不同时会怎样吗？大脑会编出一个根本不存在的节拍。这个工具让你听到它——然后你会发现，同样的声音换一套说法，感受可以截然不同。

一个很日常的提醒：感知不是被动接收，我们在主动建构自己听到的世界。
