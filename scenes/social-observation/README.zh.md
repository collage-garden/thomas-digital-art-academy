# 社会观察

*社会怎么运转，人怎么互动，文化怎么变。*

用经济学、哲学、心理学和数据做透镜，看人的世界。把观察变成理解，把理解变成作品。

## 作品

### 🎧 [双耳拍探索器——讽刺版](https://collage-garden.github.io/binaural-beats/?edition=marketing)

同一个音频引擎，三种完全不同的体验。科学版讲事实，营销版用伪科学术语和虚构的神经数据包装一模一样的声音，灵修版请出脉轮、昆达里尼和远程加持。

声音没变，变的是说法。

这是一次关于包装和感知的观察——保健品行业怎么运作，伪科学营销有多顺滑，说出来的和实际是的之间隔了多远。

**涉及知识**：经济学（忽悠怎么变现） · 心理学（安慰剂、框架效应） · 哲学（认识论、修辞与真相）
