# 托马斯数字艺术学园

### *The Collage Garden · 拼贴花园*

> 我们不教学生怎么做对，我们教他们怎么低成本地犯错，怎么在不断的犯错中学习和改善自己的作品。

---

## 这是什么

一个技术人学做艺术的地方——欢迎你一起走。

不是课程平台，是一份活着的、有版本的学习记录：粗糙的草稿、走错的路、小小的发现、还在进行中的作品。这里的东西永远是*进行时*。

## 花园里长出了什么

🎧 **[双耳拍探索器](https://collage-garden.github.io/binaural-beats/)** — 花园的第一件作品。一个交互工具，让你听到大脑从两耳频率差中"编造"出来的节拍，然后展示同一段声音在科学、营销、灵修三种包装下感受完全不同。横跨四条路径：听觉艺术、互动装置、社会观察、日常生活。（[故事](stories/thomas/#chapter-2) · [作品详情](works/)）

## 谁在做

**Thomas（唐云峰）** — 工程师，开源人，永远的初学者。技术圈待了三十年（OS → 网络 → 数据库 → ML → NLP → IoT → LLM），在好几所大学和美院教过课，最后决定回头看看小时候见到的那只茶壶——把每一步都记下来。

**AI 搭档** — 从第一天起的共同创作者，按[宪章](CHARTER-ZH.md)透明标注。

## 怎么组织的

内容沿六条交叉路径生长：

🌿 **[日常生活](scenes/everyday-life/)** — 藏在日常里的艺术和科学
🔭 **[社会观察](scenes/social-observation/)** — 社会怎么运转，人怎么互动，文化怎么变
🎵 **[听觉艺术](scenes/auditory-art/)** — 声音、音乐、声学，以及「听」本身
🎨 **[视觉艺术](scenes/visual-art/)** — 图像、形态、色彩、空间，以及「看」本身
🕳️ **[互动装置](scenes/interactive-installation/)** — 技术遇上物理空间和人的参与
🎭 **[传统艺术](scenes/traditional-art/)** — 用今天的眼睛看老手艺

路径自由交叉。一个项目可以同时长在好几条路径上。

路径下面有一个 **[知识架](knowledge/)**——按学科摆放，但始终为创作服务：

> 物理 · 数学 · 经济 · 哲学 · 心理 · 生理 · 神经科学

你不是来知识架"上课"的。你来是因为手里在做东西，需要搞懂它为什么那样。拿走你要的，回去接着干。

## 内容形式

| 形式 | 说明 |
|---|---|
| **[故事](stories/)** | 创作过程本身——日记、叙事、死胡同、火花。果实底下的根 |
| **[知识](knowledge/)** | 学习碎片、参考资料、探索笔记 |
| **[作品](works/)** | 从场景学习中长出来的创作 |
| **[课程](courses/)** | 把故事、笔记和作品串起来的学习路径 |

## 从哪里开始

先读[宪章](CHARTER-ZH.md)——它是这座花园的宪法。

然后挑一条你感兴趣的路径，走进去就行。

## 参与

谁都可以参与：分享作品、改进内容、开辟新路径。请先读[宪章](CHARTER-ZH.md)，尊重其中的原则，特别是：

- 从真问题出发，别从定义开始
- 标来源，不懂的地方说不懂
- AI 参与的内容要标清楚
